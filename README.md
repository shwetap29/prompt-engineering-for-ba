## 🚦 How to Use This Repo

1. **Start in `01-basics`** if prompting is new to you
2. **Jump to any folder** that matches your current task
3. Each file follows the same structure:
   - What it is & when to use it
   - The prompt template (copy-paste ready)
   - A filled example with real BA/DA context
   - Tips to make it better
4. Open Claude, ChatGPT, or Copilot → paste the prompt → fill in your context → go

## 📌 5 Golden Rules of Prompting (Always Apply These)

1. **Give AI a role** — start with *"You are a senior BA..."* — output quality jumps immediately
2. **Be specific about format** — say *"use a table"* or *"write a numbered list"*
3. **Paste your real context** — don't describe it vaguely, paste the actual notes or data
4. **Iterate don't restart** — add *"now make it more concise"* or *"add a risks section"* to refine
5. **Always verify** — AI writes your first draft, you write the final one
